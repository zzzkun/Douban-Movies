## 这是一个移动端的豆瓣电影推荐入口
- `top250`区域为豆瓣排行top250的电影
- `北美`区域为北美票房的排行
- `搜索`区域为输入关键字搜索
- `top250`区域和`搜索`区域 在已加载电影项的前提下滑动到底部时会自动加载新的电影项
